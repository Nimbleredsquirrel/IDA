# IDA

Intro to data analysis, ML minor course. Spring 2022/2023.

## Homeworks

**hw01 — numpy**: 10 tasks using only vectorized numpy (no loops, no if). Weighted sum, close elements, block matrix, diagonal product, normalization, prevZeroMax, symmetric matrix, cosine similarity.

**hw02 — pandas**: Student scores dataset (1000 students, 10 groups). Merge, concat, groupby, pivot tables, pass/fail classification, grade assignment.

**hw03 — EDA**: German credit scoring dataset (1000 clients, 21 features). Matplotlib/seaborn plots: mirror bar charts, violin plots, 3D scatter. Manual prediction formula, ~0.648 accuracy.

**hw04 — kNN + linear regression**: KNN decision boundaries on wine dataset (1, 3, 5, 10, 15, 25 neighbors). Linear regression with Lasso feature selection, multicollinearity check, Ridge vs Lasso comparison.

**hw05 — gradient descent**: MSE loss, GD and SGD from scratch, decaying learning rate, convergence plots. Linear regression class with fit/predict. L2 regularization (MSEL2Loss), Huber loss.

**hw06 — text classification**: 11 tasks on text data. TF-IDF, bag of words, sklearn pipeline.

**hw07 — trees + random forest**: Entropy and information gain by hand. Custom decision tree implementation (find_best_split, _predict_node). Students and mushrooms datasets. Bagging, random forest hyperparameter search with cross-validation, feature importance, ROC AUC.

**hw08 — boosting**: 10 tasks. Gradient boosting theory and practice.
